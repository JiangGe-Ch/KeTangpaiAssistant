# ketangpaiAssistant

#### 介绍
基于KeTangpaiCOntentGripper实现的安卓app，实现统一抓取自己课堂派中特定课程内容并统一排序展示的功能，解决课堂派中课程
内容被课程分散，不易查看，容易错过时间的问题。
